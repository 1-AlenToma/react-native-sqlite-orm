# react-native-sqlite-orm

an orm for react-native using sqlite database.

## Installation

```sh
npm install react-native-sqlite-orm
```

## Usage


```js
import { multiply } from 'react-native-sqlite-orm';

// ...

const result = await multiply(3, 7);
```
